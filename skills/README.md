# Agent Skills

Agent skills for real engineering work.

These are skills I use to work effectively with coding agents. They focus on practical workflows, clear communication, and getting things done without unnecessary complexity.

## Summary

This collection includes:

- **Engineering skills** for remote execution, debugging, and code quality
- **Personal skills** for custom workflows and domain-specific tasks

The skills are designed to be simple, composable, and model-agnostic. Pick what you need, adapt to your workflow.
